# Frontend Setup

## Prerequisites
- Node.js 18+ and npm

## Installation

```bash
npm install
```

## Configuration (Optional)

Create `.env.local` in `frontend/`:
```
NEXT_PUBLIC_API_BASE_URL=http://localhost:8000
```

If not set, defaults to `http://localhost:8000`

## Run

```bash
npm run dev
```

App will be available at `http://localhost:3000`
